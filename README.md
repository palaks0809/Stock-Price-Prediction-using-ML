# Lending Default Risk Prediction

## Project Overview

This project develops a Machine Learning model to predict whether a borrower is likely to default on a loan.

The objective is to help financial institutions reduce lending risk, improve credit decision-making, and minimize financial losses.

## Business Problem

Lending default risk occurs when borrowers fail to meet their debt obligations. Traditional credit assessment methods rely heavily on manual analysis, which can be time-consuming and prone to errors.

Machine Learning enables automated and accurate risk assessment using historical borrower data.

* XGBoost
